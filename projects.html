Hive Board Game AI

Hive is a two player bug-themed strategic board game in which each player is trying to surround/capture their opponent's queen bee. It can be somewhat analogous to chess, as there are several different types of pieces in the game with their own rules for movement. I personally find the game to be very fun and really enjoyed developing a bot for the game that people can play against. More information about Hive can be found here.

While working on this project, I was able to program the rules of the game, develop a method of scoring a game state, and implement a minimax algorithm for the AI. An in-depth explanation for how I implemented Hive and my minimax AI can be found here.

Making Melodious Music From Fractals

During my sophomore year at Christopher Newport University (CNU), I took a class titled Fractals and Infinity. This course delved into the mathematics and theoretical aspects of fractals. At the end of the course, we each had to write a report on a topic of our choice in relation to the class. I chose to research if musical melodies could be formed based on fractal shapes, and if so, what they would sound like.

While researching this topic, I found that there methods had been discovered previously to accomplish this. I decided to build upon this, and develop a Python program to construct the fractal melodies using the algorithms I found, and output them into a sound file.

The research paper I wrote on this topic can be found here. It discusses how fractals can be converted into musical melodies, and explains how I converted this process into a Python script.

Automated Course Registration System

This project was developed to automate CNU's course registration process for myself (or any other student that chooses to use it). To register for classes at CNU, you need to log into the registration system during your time slot, enter the course ID numbers you want to register for, then submit the form. The issue with this, is that you need to be extremely fast if you want to get into a course with very few open seats. This is because there are many other students with the same registration time that will be trying to register for the same class. This was a problem I decided to solve through the use of a Python script.

The Python script I made used the Selenium package to control a web browser and automatically register a student for their classes as soon as their time slot opened. This meant that the courses would be registered as soon as they possibly could, and left little to no room for any human error. It included a configuration file for the course ID numbers, and had the user input their student ID and password. It also was developed so that it could be run overnight as well. That way the student using the program would no longer need to wake up extremely early in the morning.

This program made course registration significantly easier for me while I was at CNU. Granted, it did take a bit of trial and error to get it to work perfectly, but I am still happy with how it turned out and I found it to be extremely useful.

The source code for this project can be found here.
